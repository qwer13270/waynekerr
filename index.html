Our Prodcuts

5400 Machine

5400 is a super machine that combines LCR meter with the Impedance Analyzer. It carries better performance specifications and feature the capabilities as the combined WK3260/WK6440. It is 3 to 5 times faster than the 3260 and could could measure up to 10 MHz.

43200 LCR Meter

43200 is a cost effective model of LCR meters at Mega Hertz level of machines. It is 30% cheaper, faster, and more precise than previous LCR meters. It can now measure up to 2MHZ, and the DC internal bias has also been enhanced to Â±40V. New upgrades

4900 LCR Engine

4900 is alternative cost effective model of 43200 with only half the price. It comes without a monitor, opening the possibility for users to customize their preferred interface within the provided command options. Compared to previous LCR Meter, it's a real bargain.

WK Data Logging

WK Data Logging is a software interface that connect with Wayne Kerr instruments. It is simple, yet powerful in running tests and retrieving test result. There are crucial features from setting, measuring, to exporting, plus guage and analysis mode switching modes
