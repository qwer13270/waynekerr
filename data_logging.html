Overview

WK Data Logging is a software interface that connect with Wayne Kerr instruments. It is simple, yet powerful in running tests and retrieving test result. There are crucial features from setting, measuring, to exporting, plus guage and analysis mode switching modes

Two mode enables different functions and supports to be done by connecting with machine

Dashboard mode functions

  • Single measurement
  • Repeat measurement
  • Save measurement data (USB key required)
  • Scan 2D barcode, new incremental measurement conditions, material information

Analysis mode supports

  • Multiple sets of scan conditions
  • The way to draw a picture is to present the measured values
  • Frequency, voltage, current, time, and temperature sweep point (Sweep)
  • Compare differences under the same measurement conditions.
